# una-ihcux-lista3
Aplicação da 1ª Heurística de Nielsen

Este projeto de Scanner foi desenvolvido com base na 1ª heurística de Jakob Nielsen:
“Visibilidade do status do sistema”.

🎯 Objetivo

Garantir que o usuário sempre saiba o que está acontecendo durante o uso do aplicativo, principalmente no processo de digitalização.

⚙️ Aplicação no Projeto

No sistema de scanner, essa heurística foi aplicada através de:

Indicadores visuais de progresso (ex: carregando, escaneando, concluído)
Mensagens de feedback imediato após ações do usuário
Avisos de erro claros quando algo dá errado
Confirmações de sucesso ao finalizar uma digitalização
✅ Benefícios
Melhor experiência do usuário
Redução de dúvidas durante o uso
Maior confiança no sistema
Navegação mais intuitiva
